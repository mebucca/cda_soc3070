Ronda de Penales - Simulacin Monte Carlo

Ronda de Penales

En esta seccin, simularemos una ronda de penales entre Brasil y Chile.

  • Primeros 5 penales: \(\mathbb{P}(\text{Gol Brasil}) = 0.8\) y \(\mathbb{P}(\text{Gol Chile}) = 0.75\)

  • Despus del 5to penal: \(\mathbb{P}(\text{Gol Brasil}) = \mathbb{P}(\text{Gol Chile}) = 0.5\)

penales

Definicin de la situacin

Para realizar la simulacin, consideramos dos casos:

1. Si no hay empate en los primeros cinco tiros:

  • Goles Brasil = variable binomial(n=5, p=0.8)

  • Goles Chile = variable binomial(n=5, p=0.75)

2. Si hay empate en los primeros cinco tiros:

  • Goles Brasil = variable binomial(n=5, p=0.8) + variable bernoulli(p=0.5)

  • Goles Chile = variable binomial(n=5, p=0.75) + variable bernoulli(p=0.5)

Luego, determinamos el ganador:

  • Si Goles Brasil > Goles Chile \(\to\) Gana Brasil

  • Si Goles Brasil < Goles Chile \(\to\) Gana Chile

Simulacin Monte Carlo

En esta seccin, definimos la funcin para realizar una simulacin de una ronda de penales. La funcin ronda_penales realiza los lanzamientos y determina el ganador segn las reglas establecidas.

library("tidyverse")
library("knitr")

p_chile  = 0.75
p_brasil = 0.8

ronda_penales <- function() {
  goles_chile  <- rbinom(1, size=5, prob=p_chile)
  goles_brasil <- rbinom(1, size=5, prob=p_brasil)
  
  # Si los dos equipos estn empatados despus de los primeros cinco tiros, 
  # pasan gol de oro
  while (goles_chile == goles_brasil) {
    goles_chile  <- goles_chile  + rbinom(1, size=1, prob=0.5)
    goles_brasil <- goles_brasil + rbinom(1, size=1, prob=0.5)
  }
  
  return(c(chile = goles_chile, brasil = goles_brasil))
}

Ejecucin de la simulacin

En esta parte, ejecutamos la simulacin para una ronda de penales y mostramos el resultado. Dependiendo del resultado, se incluir una imagen correspondiente al pas ganador.

resultados <- t(replicate(n=1, ronda_penales())) %>% as_tibble() %>%
  mutate(gana = if_else(chile > brasil, "Chile", "Brasil"))

Multiverso

En esta seccin realizamos la situacin hipottica de jugar el partido 10,000 veces bajo las mismas circunstancias. Es decir, ejecutamos la simulacin 10,000 veces para obtener una estimacin de la probabilidad de que cada equipo gane.

resultados_multiverso <- t(replicate(n=10000, ronda_penales())) %>% as_tibble() %>%
  mutate(gana = if_else(chile > brasil, "Chile", "Brasil"))

knitr::kable(prop.table(table(resultados_multiverso$gana)), col.names = c("Pas", "Probabilidad de Ganar"))
